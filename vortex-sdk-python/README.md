## Development

This project uses Poetry for dependency management and packaging.

1.  Install Poetry: `pip install poetry`
2.  Install dependencies: `poetry install --with dev`
3.  Activate virtual environment: `poetry shell`

Run tests:
```bash
pytest
```

Lint and format:
```bash
ruff check .
ruff format .
```

Type check:
```bash
mypy vortex_sdk
